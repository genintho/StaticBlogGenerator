Dear Wordpress, it is time for me to move away from you. It is has been a good and long ride but I have to go to some other solution.

But do not worry, we are still going to be friend. I really like you. You are free, easy to setup, easy to use, extremely powerful and we can do everything we want with you. That's just awesome.

But that's not for anymore, or to be more precise, not for my blog. You are an amazing products for most of the people, but you are too fat for this blog.

I know that I am picky, but I am a developper, so my need are not the need of most people.

  • I know how to write HTML and CSS so the preview/HTML editor is not necessary
  • I want to position my image precisely, and in a different way for each post so the "media" feature is not necessary
  • I don't want to customize it, I just want to write and control the visual. So no need for plugins.
  • It should allow me to write post specific JavaScript.
  • It should allow any kind of markup: I am the one in control.
  • The editor you use should display only the text, and be able to live render it.
  • It should not have 300 options that I can barely understand.
  • It should be amazingly fast to load and naviguate.
  • It should require no update.
  • It does not need an ACL system.
  • It should not require me to learn a new language.
  • It should be secure.
  • It should be easy to backup
  • It should be easy to deploy
  • It should be cheap to host and easy to scale ( like I am going to need it ).

It should be a static web site.

Well, it is a little more than that.
